# Estimacion-del-Costo-Social-del-CO2-bajo-el-enfoque-de-Perdidas-y-Da-os-de-Cambio-Climatico


"Este proyecto implementa una valoración económica de impactos ambientales basada en el modelo de Burke et al. (2023) para el cálculo del Costo Social del Carbono (SC-CO2). El algoritmo proyecta daños climáticos futuros aplicando tasas de descuento variables (5%, 3% y 2.5%), permitiendo generar escenarios comparativos ('Conservador' vs. 'Ético') que monetizan la responsabilidad intergeneracional de las emisiones de gases de efecto invernadero."

"Complementariamente, la herramienta integra el análisis de vulnerabilidad territorial propuesto por Fernandez et al. (2015) para ajustar el impacto local. Mediante la ponderación de factores de Exposición, Sensibilidad y Capacidad de Adaptación, el sistema discrimina el riesgo según la ubicación del proyecto, asignando una carga económica mayor a infraestructuras situadas en ecosistemas frágiles con baja resiliencia, como la región Amazónica."
